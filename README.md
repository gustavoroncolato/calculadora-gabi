# Calculadora de Drogas Vasoativas

Calculadora de conversão de dose para drogas vasoativas utilizadas em UTI.  
Converte **ml/h ↔ mcg/kg/min** (e **UI/h** para vasopressina) de forma rápida e prática.

## Drogas disponíveis

| Droga | Apelido | Diluição |
|---|---|---|
| Noradrenalina | Nora | 2 amp (1mg/ml, 4ml) + 92ml SG — também disponível dobrada |
| Dobutamina | Dobuta | 2 amp (12,5mg/ml, 20ml) + 60ml SF |
| Nitroprussiato | Nipride | 1 amp (25mg/ml, 2ml) + 250ml SG 5% |
| Trinitrato de Glicerila | Tridil | 1 amp (5mg/ml, 5ml) + 100ml SG |
| Vasopressina | Vaso | 1 amp (20UI/ml, 1ml) + 99ml SF 0,9% |

## Funcionalidades

- Conversão bidirecional: ml/h → dose e dose → ml/h
- Modo **Padrão HR** com protocolos pré-configurados
- Modo **Cálculo Manual** para diluições customizadas
- Versão **offline (PWA)** instalável em qualquer dispositivo
